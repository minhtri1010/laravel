Tabs

A single content area with multiple panels, each associated with a header in a list.

Dependencies

Javascript

  • Bootstrap tabs core - /assets-minified/widgets/tabs/tabs.js

CSS

  • /assets-minified/elements/menus.css - Contains the CSS needed for the bootstrap tabs structure. Styling is achieved from the color theme CSS file (/assets-minified/themes/../*.css).

To see all the available options & configurations, we suggest reading the Tabs external documentation accessible through the link below.

jQueryUI Tabs

Use the examples below to create jQueryUI powered tabbed content.

Dependencies

Javascript

  • jQuery Core - /assets-minified/js-core/jquery-core.js
  • jQueryUI Core - /assets-minified/js-core/jquery-ui-core.js
  • jQueryUI Widget - /assets-minified/js-core/jquery-ui-widget.js
  • jQueryUI Tabs - /assets-minified/widgets/tabs-ui/tabs.js

CSS

  • /assets-minified/widgets/tabs-ui/tabs.css - Contains the CSS needed for the tabs structure. Styling is achieved from the color theme CSS file (/assets-minified/themes/../*.css).

To see all the available options & configurations, we suggest reading the Tabs external documentation accessible through the link below.
