Wizard

Wizards allow you to go through different steps of content using a navigation or pager menu.

Dependencies

Javascript

  • jQuery core - /assets-minified/js-core/jquery-core.js
  • Wizard - /assets-minified/widgets/wizard/wizard.js
  • Wizard demo - /assets-minified/widgets/wizard/wizard-demo.js

CSS

  • /assets-minified/widgets/wizard/wizard.css - Contains the CSS needed for the Wizard structure. Styling is achieved from the color theme CSS file (/assets-minified/themes/../*.css).

To see all the available options & configurations, we suggest reading the Wizard external documentation accessible through the link below.
