Dependencies

Javascript

  • jQuery Core - /assets-minified/js-core/jquery-core.js
  • Owl Carousel - /assets-minified/widgets/owlcarousel/owlcarousel.js
  • Owl Carousel demo - /assets-minified/widgets/owlcarousel/owlcarousel-demo.js

CSS

  • /assets-minified/widgets/owlcarousel/owlcarousel.css - Contains the CSS needed for the Owl Carousel structure. Styling is achieved from the color theme CSS file (/assets-minified/themes/../*.css).
