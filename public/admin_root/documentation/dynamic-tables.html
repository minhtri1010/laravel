Dynamic tables

Our framework comes packed with a lot of table options variations.

Dependencies

Javascript

  • jQuery Core - /assets-minified/js-core/jquery-core.js
  • jQuery Datatables - /assets-minified/widgets/datatable/datatable.js
  • jQuery Datatables bootstrap - /assets-minified/widgets/datatable/datatable-bootstrap.js
  • Datatables bootstrap - /assets-minified/widgets/datatable/datatable-bootstrap.js

CSS

  • /assets-minified/widgets/datatable/datatable.css - Contains the CSS needed for the datatable structure. Styling is achieved from the color theme CSS file (/assets-minified/themes/../*.css).
